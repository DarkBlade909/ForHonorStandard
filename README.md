# For Honor Standard
A Unity shader to replicate For Honor's customizable gear shader

<img width="533" height="514" alt="image" src="https://github.com/user-attachments/assets/ec7be586-8367-44b2-a959-1b9d3b6dc42c" />

## REQUIREMENTS
- z3y's Graphlit: https://github.com/z3y/Graphlit
## Features
- Packed ORM Maps
- Detail Maps
- Clearcoat
- Anisotropy
- Geometric Specular Anti-aliasing
- Indirect Specular
- Specular Occlusion
- Backface Normals
- Green Channel Flip Toggles
